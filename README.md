# Booking Auth Service (Микросервис авторизации)

Микросервис для аутентификации и управления пользователями в системе бронирования. Построен на базе **FastAPI**, **SQLAlchemy** и **Alembic**.

## 🚀 Технологический стек
* **Python** 3.10+
* **FastAPI** — веб-фреймворк
* **SQLAlchemy** — ORM для работы с базой данных
* **Alembic** — управление миграциями БД
* **SQLite** — база данных (для локальной разработки)

---

## 🛠️ Инструкция по локальному запуску

### 1. Клонирование репозитория
```bash
git clone git@github.com:Ramazan101/booking-auth-service.git
cd booking-auth-service
```

### 2. Настройка виртуального окружения
Создайте и активируйте виртуальное окружение:

**Для Windows (PowerShell):**
```powershell
python -m venv .venv
.venv\Scripts\Activate.ps1
```

**Для Linux / macOS:**
```bash
python3 -m venv .venv
source .venv/bin/activate
```

### 3. Установка зависимостей
```bash
pip install -r req.txt
```

### 4. Применение миграций базы данных
Создайте локальную базу данных SQLite и примените существующие миграции Alembic:
```bash
alembic upgrade head
```

### 5. Запуск сервера разработки
```bash
uvicorn main:app --reload
```
Сервер будет доступен по адресу: `http://127.0.0.1:8000`

## 📄 Документация API

После запуска проекта интерактивная документация (Swagger UI) доступна по ссылке:
👉 `http://127.0.0`

### Основные эндпоинты (разрабатываемые/готовые):
* `POST /api/auth/register` — Регистрация нового пользователя
* `POST /api/auth/login` — Аутентификация и получение токена
* `GET /api/user/me` — Получение профиля текущего пользователя
